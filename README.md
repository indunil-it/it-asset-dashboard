# IT Asset Inventory Dashboard 💻

A clean, responsive web dashboard to track and manage IT assets. Built with Vanilla JavaScript and deployed on GitHub Pages.

**🔗 Live Demo:** https://indunil-it.github.io/it-asset-dashboard/

## ✨ Features
- **Real-time Statistics:** Auto-calculates Total Assets, Active Devices, and Items In Repair.
- **Dynamic Table:** Asset data is rendered dynamically using JavaScript.
- **Responsive Design:** Works perfectly on Desktop, Tablet, and Mobile.
- **Clean UI:** Modern, dark-blue theme with card-based layout.
- **Zero Dependencies:** Built with pure HTML, CSS, and Vanilla JavaScript.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Deployment:** GitHub Pages
- **Version Control:** Git & GitHub
